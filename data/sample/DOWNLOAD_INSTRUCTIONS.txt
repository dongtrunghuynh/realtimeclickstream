REES46 E-Commerce Dataset — Download Instructions
==================================================

Dataset URL:
  https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store

Steps:
  1. Create a free Kaggle account at https://www.kaggle.com
  2. Navigate to the dataset URL above
  3. Click the "Download" button (top-right)
  4. Download "2019-Oct.csv" — approximately 2GB compressed
  5. Extract and place the CSV file here:
       data/sample/2019-Oct.csv

Alternative (Kaggle CLI):
  pip install kaggle
  kaggle datasets download -d mkechinov/ecommerce-behavior-data-from-multi-category-store
  unzip ecommerce-behavior-data-from-multi-category-store.zip -d data/sample/

Verify the download:
  wc -l data/sample/2019-Oct.csv
  # Expected: ~42,448,764 lines

File is in .gitignore — NEVER commit it (2GB, contains no PII but is too large for git).

Column reference:
  event_time    ISO8601 timestamp of the user action
  event_type    view | cart | purchase
  product_id    Product identifier
  category_id   Numeric category ID
  category_code Human-readable category path (e.g. electronics.smartphone) — nullable
  brand         Brand name — nullable
  price         Product price in USD
  user_id       User identifier
  user_session  Session ID (used as session_id in this pipeline)
